# Aplicación "to do List"
## Entrega final para la materia PWA

---- 

### Descripción
Esta es una aplicación tipo To-Do List, inspirada en la simplicidad de Notion, que permite gestionar tareas de forma intuitiva.
Incluye un CRUD completo, lo que significa que podés:

- Crear nuevas tareas

- Leer y visualizar la lista actual

- Actualizar su estado (completada / pendiente)

- Eliminar tareas cuando ya no las necesitás

- Diseñada para ser rápida, simple y práctica.


### Instalación

1. git clone https://github.com/sofi-ne-1994/todolistFinal.git
2. carpeta todolist(backend)
    - pip install -r requirements.txt
    - cd todolist
    - python manage.py runserver
    
3. carpeta todolistFront(front)
    - cd todolistFront
    - npm install
    - npm run dev
